## 🚀 快速启动

### 方法1: 使用启动脚本（推荐）

**Windows:**
```bash
cd gui
start_gui.bat
```

**Linux/Mac:**
```bash
cd gui
chmod +x start_gui.sh
./start_gui.sh
```

### 方法2: 手动启动

```bash
# 1. 进入GUI目录
cd gui

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动GUI
streamlit run app.py
```

### 访问界面

启动后，浏览器会自动打开：
```
http://localhost:8501
```

## 🎯 使用示例

### 示例1: 运行均线交叉策略回测

1. 启动GUI
   ```bash
   cd gui
   streamlit run app.py
   ```

2. 前往"策略回测"页面

3. 配置参数：
   - 策略：选择"均线交叉 (MA Cross)"
   - 开始日期：2024-01-01
   - 结束日期：2024-12-31
   - 股票代码：sh.600519（贵州茅台）
   - 初始资金：1,000,000
   - MA5: 5
   - MA20: 20

4. 点击"🚀 开始回测"

5. 等待回测完成（显示进度条）

6. 查看结果：
   - 总收益率
   - 夏普比率
   - 资金曲线图
   - 详细指标

### 示例2: 配置AI策略

1. 前往"系统配置"页面

2. 配置DeepSeek API Key

3. 保存配置

4. 前往"策略回测"页面

5. 选择"AI驱动策略"

6. 运行回测

## 💡 使用技巧

### 1. 提高回测速度
- 使用日线数据代替分钟数据
- 减少回测时间范围
- 限制股票数量

### 2. 优化策略参数
- 使用"策略管理"页面保存多个版本
- 记录每次参数变更
- 对比不同参数的回测结果

### 3. 分析性能
- 关注风险调整后的收益（夏普比率）
- 注意最大回撤
- 分析交易次数和胜率

## 🐛 问题排查

### GUI无法启动
```bash
# 检查Python版本
python --version  # 需要3.8+

# 重新安装依赖
pip install -r requirements.txt --force-reinstall

# 查看错误日志
streamlit run app.py --logger.level=debug
```

### 回测失败
1. 检查股票代码格式
2. 确认日期范围合理
3. 查看控制台错误信息

### 图表显示异常
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 更换浏览器
