## 🚀 使用方法

### 启动GUI

**Windows:**
```bash
cd D:\ApexQuant\gui
start_gui.bat
```

**Linux/Mac:**
```bash
cd /path/to/ApexQuant/gui
chmod +x start_gui.sh
./start_gui.sh
```

**手动启动:**
```bash
cd gui
pip install -r requirements.txt
streamlit run app.py
```

### 访问界面

浏览器自动打开或手动访问：
```
http://localhost:8501
```

## 📝 使用示例

### 示例1: 运行回测

1. 启动GUI
2. 点击侧边栏"📊 策略回测"
3. 配置参数：
   - 策略：均线交叉
   - 日期：2024-01-01 to 2024-12-31
   - 股票：sh.600519
   - 初始资金：1,000,000
4. 点击"🚀 开始回测"
5. 查看结果和图表

### 示例2: 配置系统

1. 点击侧边栏"⚙️ 系统配置"
2. 配置API密钥
3. 设置交易参数
4. 配置风控规则
5. 点击保存
